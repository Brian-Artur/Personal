- # Renderizado
	- **Render** es la *acción* de que hace que React **ejecute tu función componente**. (la llama, corre de arriba abajo). Es decir que se lee de arriba a abajo.
	- El **return** es el resultado esa ejecución, que *produce*: el JSX.
